library(ggplot2)

## Warning: package 'ggplot2' was built under R version 2.15.2

In this scatter plot, we see the trend of obese people having highest abdomen size and neck size, followed by overweight people, healthy people, and underfat people. 

g <- ggplot(measures, aes(Abdomen, Neck, ))

g + geom_point(aes(color = FatStat), size = 4, alpha = 1/2) + labs(title = "Neck and Abdomen Size by Fat Status") + 
    labs(x = ("Abdomen size in inches"), y = "Neck size in inches")

When looking at this scatter plot, we see that age is pretty even distributed. 

g + geom_point(aes(color = Age), size = 4, alpha = 1/2) + labs(title = "Neck and Abdomen Size by Age Groups") + 
    labs(x = ("Abdomen size in inches"), y = "Neck size in inches")

These facets show the distributions of neck size and abdomen size in the 4 different fat status categories and the 4 age groups. 

g + geom_point() + facet_grid(~FatStat) + labs(title = "Neck and Abdomen Size By Fat Status") + 
    labs(x = ("Abdomen size in inches"), y = "Neck size in inches")
